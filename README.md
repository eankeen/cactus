# cactus

## Description

Meant to be used as as a secondary application launcher. As an alternative a separate launch mode you can create
with i3, etc.

Ex. sxhkdrc

```txt
super + y
	cactus
```

## Configuration

```toml
# $XDG_CONFIG_HOME/cactus/bindings.toml
[E]
cmd = "echo 'fortune'"
```
